# Учебный сайт «Круг Иттена»
Учебный проект по дисциплине «Веб-разработка».

## Тематика
Использование круга Иттена для подбора цветовых сочетаний в одежде и стиле.

## Реализовано
- Многостраничный сайт
- Google Analytics и Яндекс.Метрика
- Страница 404
- robots.txt и sitemap.xml
- Защита контента от копирования
- Отображение активности пользователей
